# Vekus Server PN Generator v2.0

Веб-приложение для автоматической генерации артикулов и описаний серверов Vekus на основе конфигурации комплектующих.

## Возможности

- Ввод конфигурации сервера через таблицу комплектующих (шасси, CPU, RAM, накопители, RAID, GPU, сеть, БП и др.)
- Автоматическое определение платформы, форм-фактора, поколения CPU, уровня RAM, типа PSU
- Генерация артикула (PN) по стандарту Vekus
- Генерация текстового описания для каталога
- Работает полностью офлайн — один HTML-файл без зависимостей

## Быстрый старт

1. Скачайте файл `vekus-generator.html`
2. Откройте в браузере (двойной клик)
3. Заполните таблицу комплектующих
4. Нажмите **СГЕНЕРИРОВАТЬ АРТИКУЛ И ОПИСАНИЕ**

## Структура артикула

```
VSC.R2U.I25.CS.1.H/408
│ │  │   │   │  │ │ └── Дата (месяц + день)
│ │  │   │   │  │ └── PSU: F=фикс, H=hot-swap, D2=2x redundant, D4=4x redundant
│ │  │   │   │  └── RAM уровень: 1=Entry(<192GB), 2=Mid(192-511GB), 3=High(512-1023GB), 4=Ultra(1024+GB)
│ │  │   │   └── Диски: [D|C|S][S|A|N|M] = bays(D=4-8,C=12,S=24+) + тип(S=SATA,A=SAS,N=NVMe,M=Mixed)
│ │  │   └── CPU: I/E + сокеты + поколение (I=Intel, E=AMD)
│ │  └── Форм-фактор: R1U, R2U, R4U, T4U (Tower)
│ └── Категория: C=Compute, G=GPU, S=Storage, E=Edge
└── Платформа: VS=Supermicro, VG=Gigabyte, VA=ASUS, VC=Chenbro, VT=TTY
```

## Поддерживаемые типы комплектующих

| Тип | Описание |
|-----|----------|
| Шасси | Серверная платформа (определяет вендор, форм-фактор, кол-во дисковых корзин, LOM) |
| Процессор | Intel Xeon (автоопределение поколения 3-6) |
| Кулер | Система охлаждения CPU |
| Память | DDR4/DDR5 ECC REG модули (автоподсчёт общего объёма) |
| Накопитель | SATA HDD, SAS HDD, SATA SSD, SAS SSD, NVMe, M.2 |
| Сетевой адаптер | Дополнительные сетевые карты (1/10/25/100 GbE) |
| Контроллер дисковых массивов | RAID контроллеры (MegaRAID, PERC, SmartRAID) |
| GPU | Видеокарты / ускорители (RTX, A100, H100 и др.) |
| Блок питания | PSU с автоопределением типа (fixed/hot-swap/redundant) |
| Прочее | ПО, мониторы, кабели и другие аксессуары |

## Формат описания

```
Сервер Vekus "Кварк" 2U 12x 3.5" в составе: 2xXeon Gold 6530 (2.10-4.00GHz/160MB/32c/270W)/
2x32Gb DDR5 ECC REG/2x480Gb SATA SSD/RAID 9560-16i/BBU/2x10GbE LOM/2x25Gb SFP28 NIC/
IPMI/2x1600W/Rails
```

- Tower серверы обозначаются как **ПЭВМ** (без Rails)
- Rack серверы — как **Сервер** (с Rails)
