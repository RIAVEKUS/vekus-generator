# Vekus Server PN Generator v2.0

Веб-приложение для автоматической генерации артикулов и описаний серверов Vekus на основе конфигурации комплектующих.

## Запустить онлайн

**[▶ Открыть Vekus Generator](https://riavekus.github.io/vekus-generator/vekus-generator.html)**

---

## Возможности

- Ввод конфигурации сервера через таблицу комплектующих
- Автоматическое определение платформы, форм-фактора, поколения CPU, уровня RAM, типа PSU
- Генерация артикула (PN) по стандарту Vekus
- Генерация коммерческого описания для каталога
- **Импорт из Excel / CSV** — загрузка конфигурации из файла с автоопределением колонок
- **Скачать шаблон Excel** — готовый файл с примерами для заполнения
- **Копирование результата для Excel** — артикул и описание вставляются в две соседние ячейки
- Работает в браузере без установки (один HTML-файл)

---

## Быстрый старт

1. Скачайте файл `vekus-generator.html`
2. Откройте в браузере (двойной клик)
3. Заполните таблицу комплектующих вручную **или** импортируйте из Excel
4. Нажмите **СГЕНЕРИРОВАТЬ АРТИКУЛ И ОПИСАНИЕ**

## Структура артикула

```
VGC.R2U.I24.EM.2.H/408
│ │  │   │   │  │ │ └── Дата (месяц + день)
│ │  │   │   │  │ └── PSU: F=фикс, H=hot-swap, D2=2x redundant, D4=4x redundant
│ │  │   │   │  └── RAM: 1=Entry(≤50 ГБ), 2=Mid(50–200 ГБ), 3=High(200 ГБ–1 ТБ), 4=Ultra(1 ТБ+)
│ │  │   │   └── Диски: [кол-во][тип] — см. таблицы ниже
│ │  │   └── CPU: I/E + сокеты + поколение (I=Intel, E=AMD)
│ │  └── Форм-фактор: R1U, R2U, R4U, T (Tower)
│ └── Категория: C=Compute, G=GPU, S=Storage, E=Edge
└── Платформа: VS=Supermicro, VG=Gigabyte, VA=ASUS, VC=Chenbro, VT=TTY
```

### Коды количества дисковых корзин

| Код | Макс. корзин |
|-----|-------------|
| C   | ≤ 12        |
| D   | ≤ 16        |
| E   | ≤ 24        |
| F   | ≤ 36        |
| G   | ≤ 48        |
| H   | ≤ 60        |

> Количество определяется по описанию шасси — суммируются все группы `NxSFF/NxLFF`.  
> Задние корзины (последняя группа ≤ 4 слотов, только SATA/SAS) в классификацию **не включаются**.

### Коды типа дисков

| Код | Тип                          |
|-----|------------------------------|
| S   | SATA / SAS                   |
| N   | NVMe                         |
| M   | Смешанная (NVMe + SATA/SAS)  |

> Тип определяется по возможностям корзин шасси из описания платформы.

## Импорт из Excel

Нажмите **↥ Импорт из Excel** и выберите файл `.xlsx`, `.xls` или `.csv`.

Колонки распознаются автоматически по следующим вариантам названий:

| Поле      | Варианты заголовка                                              |
|-----------|-----------------------------------------------------------------|
| Тип       | `Тип`, `Type`, `Категория`, `Category`                          |
| Подтип    | `Подтип`, `Subtype`                                             |
| Артикул   | `Артикул`, `Арт`, `PN`, `SKU`, `Part Number`, `Партномер`       |
| Описание  | `Описание`, `Description`, `Наименование`, `Название`, `Товар`  |
| Кол-во    | `Кол-во`, `Количество`, `Qty`, `Quantity`, `Кол`                |

Если колонка **Тип** отсутствует, тип комплектующей определяется автоматически по тексту описания (Xeon/EPYC → Процессор, DDR4/DIMM → Память, NVMe/SSD → Накопитель и т.д.).

Кнопка **↓ Шаблон Excel** скачивает готовый файл `vekus-template.xlsx` с примерами и листом-справочником допустимых типов.

## Поддерживаемые типы комплектующих

| Тип                  | Описание                                                                 |
|----------------------|--------------------------------------------------------------------------|
| Шасси                | Серверная платформа — определяет вендор, форм-фактор, кол-во корзин, LOM |
| Процессор            | Intel Xeon поколений 2–6 (Cascade Lake → Granite Rapids), AMD EPYC       |
| Кулер                | Система охлаждения CPU                                                   |
| Память               | DDR4/DDR5 ECC REG (автоподсчёт общего объёма для кода RAM)               |
| Накопитель           | SATA HDD, SAS HDD, SATA SSD, SAS SSD, NVMe, M.2                         |
| Контроллер           | RAID-контроллеры (MegaRAID, PERC, SmartRAID); +BBU при наличии батареи   |
| Аксессуары           | Rails, BBU, батареи резервного питания и прочие аксессуары               |
| Сетевой адаптер      | Дополнительные сетевые карты (1/10/25/100 GbE, Infiniband)               |
| GPU                  | Ускорители вычислений (RTX, A100, H100, Blackwell и др.)                 |
| Тип БП               | PSU с автоопределением типа (fixed / hot-swap / redundant)               |
| Операционная Система | Astra Linux, РЕД ОС, Windows Server и другие ОС                         |
| Прикладное ПО        | Kaspersky, Dallas Lock и другое прикладное ПО                            |
| Прочее               | Мониторы, кабели и прочее (не включается в коммерческое описание)        |

## Формат описания

```
Сервер Vekus "Кварк" 2U в составе: 2x Intel Xeon Gold 6348 (2.60-3.50GHz/42MB/28cores)/
8x32GB DDR4/4x1TB NVMe SSD/2x1.6TB SAS SSD/RAID 9560-16i/2x25GbE SFP28/IPMI/2x2000W/RAILS
```

- Rack-серверы обозначаются как **Сервер**, оканчиваются на `/RAILS`
- Tower-серверы обозначаются как **ПЭВМ**, без `/RAILS`
- RAID с батареей: `9560-16i/BBU`
- Тип `Прочее` и `Аксессуары` (кроме BBU и RAILS) в описание **не включаются**
